# SmartHotel
A modular Hotel Management System built with ASP.NET Core Web API, Angular, and SQL Server.

# Overview
HotelDesk is a full-stack application designed to simplify hotel management. The system is built using a modular architecture, making it easy to extend with new features and adapt to different hotel requirements.
The application is intended to support daily hotel operations, including room management, reservations, guest records, staff administration, and additional hotel services.
